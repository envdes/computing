Create a website using Sphinx

For each project, we’ll create a website using Sphinx and Markdown, mainly for two reasons:

  • serving as a demo for the community

  • reminding yourself

You can either use Read the Docs (with EthicalAds) or GitHub Actions (under your own domain and ad-free) to build

Install Sphinx

# create an environment with the name "docs"
conda env create -n docs python=3.8
conda activate docs
# install sphinx
conda install -c conda-forge sphinx

Use Sphinx

cd /path/to/project
mkdir docs
cd docs
sphinx-quickstart

Follow the options:

Separate source and build directories (y/n) [n]: n
Project name: YourProjectName
Author name(s): YourName
Project release []: v0.0.0
Project language [en]: en

Follow the template HERE to modify the index.rst and add new pages.

Follow the template HERE to set up main.yml for GitHub Actions.
